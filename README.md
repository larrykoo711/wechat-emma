# wechat-emma（wxemma）

一个 macOS 上的效率小工具。如果你有多个微信需要同时登录（比如一个生活号、一个工作号），
`wxemma` 可以帮你在同一台 Mac 上同时打开多个互相独立的微信，各自登录、互不影响。

## 它能做什么

- 一条命令新增一个独立的微信
- 查看、启动、删除已经创建的微信
- 微信升级后一键重建，保持版本一致
- 中英文界面自动切换

## 安装

需要先装好官方微信。然后在「终端」里执行：

```
brew install larrykoo711/tap/wxemma
```

## 常用命令

| 你想做的事 | 输入的命令 |
| --- | --- |
| 新增一个微信 | `sudo wxemma add` |
| 查看已有的微信 | `wxemma list` |
| 启动全部微信 | `wxemma open` |
| 删除某一个 | `sudo wxemma remove` |
| 微信升级后重建 | `sudo wxemma rebuild` |

删除时会先列出现有的微信让你选编号，并询问是否一并清除该微信的聊天记录，按需选择即可。

## 配合 AI 助手使用

本工具附带一个技能，可以让 Claude 等 AI 助手直接帮你操作。通过 [skills.sh](https://skills.sh)
一条命令即可安装：

```
npx skills add larrykoo711/wechat-emma
```

## 使用提醒

- 每个新微信首次使用需要扫码登录一次。
- 如果多个微信对同一个快捷键都有反应，可在各自的「设置 → 快捷键」里关闭全局快捷键。
